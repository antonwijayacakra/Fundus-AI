Hasil akurasi dan validasi akurasi Mobilenet v2
Plot tersebut menunjukkan akurasi model dan akurasi validasi selama 100 epoch pelatihan. Akurasi pelatihan (garis biru) meningkat stabil hingga mencapai sekitar 95%, sedangkan akurasi validasi (garis oranye) meningkat hingga sekitar epoch ke-20 lalu berfluktuasi antara 85% hingga 90%. Perbedaan signifikan antara akurasi pelatihan dan validasi setelah epoch ke-20 mengindikasikan potensi overfitting, di mana model bekerja sangat baik pada data pelatihan tetapi kurang menggeneralisasi dengan baik pada data baru.

Hasil loss pelatihan dan loss validasi mobilenet v2
Plot tersebut menunjukkan nilai loss model dan loss validasi selama 100 epoch pelatihan. Nilai loss pelatihan (garis biru) menurun secara konsisten, menunjukkan bahwa model semakin baik dalam meminimalkan kesalahan pada data pelatihan, mencapai sekitar 0.1 pada epoch ke-100. Sementara itu, nilai loss validasi (garis oranye) menurun pada awal pelatihan tetapi kemudian berfluktuasi antara 0.3 hingga 0.4 setelah sekitar epoch ke-20. Fluktuasi dan perbedaan yang signifikan antara nilai loss pelatihan dan validasi menunjukkan bahwa model mungkin mengalami overfitting, di mana model belajar terlalu baik pada data pelatihan namun kurang mampu menggeneralisasi pada data baru.
